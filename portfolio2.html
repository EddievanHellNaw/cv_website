MTG Collection App

Project done as a personal project to organize and possibly upgrade to a small e-commerce site for a very well-known collectible card game.

The objective was to develop a friendly interface for people interested in buying cards from a personal collection. Each item shows relevant information for collector such as: Card condition, name, quantity and price.

The price utilizes a simple web scrapping algorithm to update the prices a certain day of the week.

Technologies used: HTML, CSS, DJANGO, PYTHON, HEROKU, WEB SCRAPPING
